# 🎨 Art-Gallery

Art-Gallery — это серверное приложение, предназначенное для каталогизации и управления коллекцией произведений искусства. Система позволяет хранить информацию о картинах, художниках и обеспечивает удобный доступ к данным через REST API. Приложение может использоваться как для личных коллекций, так и для небольших галерей, нуждающихся в цифровизации своего фонда.

## 🌟 Основные функции

1. **🖼️ Управление произведениями искусства:**
   - Добавление, редактирование и удаление информации о картинах.
   - Хранение детальной информации: название, год создания, техника исполнения, размеры, изображение.
   - Просмотр списка всех картин с возможностью пагинации.

2. **👨‍🎨 Управление художниками:**
   - Создание и редактирование профилей художников (имя, биография, годы жизни, страна).
   - Связывание картин с конкретными авторами.

3. **🔍 Поиск и фильтрация:**
   - Поиск картин по художнику, названию или году создания.
   - Фильтрация коллекции по различным критериям (например, все картины конкретного художника).

4. **📂 Категоризация:**
   - Группировка произведений по жанрам, стилям или периодам.
   - Назначение нескольких категорий одной картине.

5. **📡 REST API:**
   - Полноценный API для интеграции с веб-интерфейсом или мобильным приложением.
   - Стандартизированные форматы запросов и ответов.

## 🛠️ Технологии

- **Язык программирования:** Java 
- **Фреймворк:** Spring Boot 3.x
- **База данных:** PostgreSQL (или MySQL)
- **Сборка:** Maven
- **Документация API:** Swagger/OpenAPI
